# Participant Preparation

{% stepper %}
{% step %}
### Pre-visit

10mins prior to scan session meet participant at Scannexus reception (see corona specific guidelines)
{% endstep %}

{% step %}
### Move to the waiting room

* [ ] Complete forms
* [ ] Fill disposable electrodes with gel and attach 2 to palm of left hand, and 2 to wrist of left hand
* [ ] Ask participant if they need to use bathroom, or remove under-wired bra
{% endstep %}

{% step %}
### Move to the control room

* [ ] Ask participant to remove other metal objects, shoes
* [ ] Check weight and height if unsure
{% endstep %}

{% step %}
### Move to the MRI scanner

* [ ] Give earplugs
* [ ] Place participant supine and pad head, attach tape across forehead
* [ ] Knee cushion and Velcro strap for button boxes
* [ ] Place button box (buttons 1 and 2) under index and middle finger of right hand
* [ ] Place emergency squeeze ball in right hand
* [ ] Give blanket if wanted
* [ ] Attach electrode leads
* [ ] Place head coil (64 channel) and isocenter
{% endstep %}
{% endstepper %}
