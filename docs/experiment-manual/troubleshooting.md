# Troubleshooting



{% hint style="info" %}
## No shocks delivered?

* Check DS7 turned on, safety switch off, red light not on, mA > 0
* TTL from stim PC to DS7 properly connected (parallel port splitter labelled ‘DS7’ to ‘in’ on back of DS7)?
* Banana plug on front of DS7 to filter plate?
* Filter plate connection solid?
* Electrodes properly filled with gel? Clips properly attached?
{% endhint %}

{% hint style="info" %}
## No SCR signal / Acqknowledge not working?

* Check Acqknowledge dongle plugged in? (try other USB port)
* BIOPAC turned on?
* Ethernet cable properly connected (ACQ PC to BIOPAC)?
* Cable from BIOPAC to filter plate?
* Filter plate connection solid?
* Electrodes properly filled with gel? Clips properly attached?
{% endhint %}

{% hint style="info" %}
## PsychoPy not detecting triggers or button presses

* Be sure first dummy scans are over (approx. 10s)
* Keyboard of Stim PC set to ENG (bottom right corner)?
* Response box switch set to ‘PC2’, piezo boxes set to ‘USB’?
* USB trigger cable plugged in to Stim PC docking station?
{% endhint %}

{% hint style="info" %}
## Participant can’t see visual stimuli or images reversed

* Turn on projector from scanner stim PC?
* Attach DisplayPort cable to Stim PC docking station?
* Projector video switch set to ‘laptop’?
* Projector screen properly placed?
* Mirror on head coil right way around?
{% endhint %}
